Introduction

This is my area of my site, dedicated to Robotics and Artificial Intelligence. I have built bots using Lego Mindstorm Technology since 2003 when I bought my first kit, RCX 1.5. RCX brick is a excellent tool to learn and know the robotic's world.

I decided to create this section, because in 2006, I bought my new Lego Mindstorm Kit, named NXT. New Lego Mindstorm brick, NXT, improve the features in relation to RCX brick and it includes Bluetooth connection. This feature, BT, adds new possibilities to create "Social Agents" or complex entities using two or more NXT bricks to be a single robotic entity. Besides, this technology allow to build agents controlled by Palm PDA or another PDA system to manage the Intelligence of the system.

Goals

The goals of this project are:

  • Improve my knowledge in A.I.
  • Artificial Vision using NXT
  • Create a game using NXT technology emulating Battletech
  • Generate ASIMOV laws in NXT brick
